# Agent Projects

A collection of AI-powered agent applications and real-world use cases built using Large Language Models (LLMs), Agentic AI patterns, and modern AI tooling.

This repository serves as a playground for experimenting with autonomous agents, document processing, conversational AI, workflow automation, and enterprise AI solutions.

## Overview

The goal of this repository is to showcase practical implementations of Agentic AI concepts, including:

* Conversational AI applications

* Retrieval-Augmented Generation (RAG)

Each project is designed to solve a specific business problem while demonstrating modern AI engineering practices.

## Getting Started

### Prerequisites

* Python 3.10+
* UV
* Ollama
* Git

### Clone the Repository

```bash
git clone https://github.com/<your-username>/agent-projects.git
cd agent-projects
```

### Navigate to a Use Case

Example:

```bash
cd agentusecases/pdf-scanner
```

### Install Dependencies

```bash
uv sync
```

### Run the Application

```bash
uv run app.py
```
